Human Trafficking Data Set

Project Overview

This project was a colaberative effort between myself and my classmate. The overall goal of this project was to analyze a data set containing information about human trafficking incidents around the world. We aimed to identify patterns and trends in the data to better understand the scope and nature of human trafficking globally.
